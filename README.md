# Student Result Management System

A menu-driven Python application for managing student academic records.

## Features

- Add Student Records
- Add Subject Marks
- Calculate Percentage
- Grade Calculation
- Search Student
- Generate Report Card
- Rank Students
- Save Data to JSON
- Load Data Automatically

## Concepts Used

- Object-Oriented Programming
- Functions
- Lists
- Dictionaries
- JSON Handling
- File Handling
- Sorting
- Searching

## Run

```bash
python main.py
